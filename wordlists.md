# Словарная анкета для работы в Микрякове-2016

## Необходимые пояснения
Порядок слов примерно соответствует словарю-минимуму из Кибрик-72 (Приложение 5), дополнения продиктованы здравым смыслом. До тире выписан русский перевод. Перед чертой даны слова из горномарийского словаря Саватковой (2008), после черты, при необходимости, то есть, в тех случаях, когда есть отличия, даны слова из словаря северо-западных говоров Иванова и Тужарова (1971). Глаголы даются в диагностической форме NPST.3SG, остальное по традиции.
